What does "Full" in Full Stack mean?P(NTS: add illustration)

			Full means that you develop both the front end, and the back end.
			The front end is what the user sees on what is called the client computer.
			The back end is what is on the server computer that delivers content to the front end.

What does "Stack" in Full Stack mean?(NTS: add illustration)

			Stack means that several programs, a "stack" of programs is used.
			On the front end I use HTML5, CSS3, and JavaScript.
			On the back end I use PHP, and MySQL.
			And I also have a LAMP stack on my server computer.

What does "LAMP Stack" mean?(NTS: add illustration)

			LAMP is a stack used for our web page server - remember, a server sends web pages to a client.
			LAMP stands for:
			L = Linux, which is the basis of our operating system
			A = Apache, which is our web server
			M = MySQL, the database we use (click here for a visual)
			P = PHP, which is the programming language we use on the server side

1. Explain Database: Organized or structured way to store information in a safe way. allows easy retrieval, updating, analysis, and output of data. Used by many businesses

2. Explain MySQL: MySQL is a open source relational database management system (RDBMS). MySQL allows the user to be able to create, update, and administer a relational database. MySQL stores data in the form of related tables. The program is one of the most popular databases used.

3. Explain SQL: SQL is a Structured Query Language. SQL is used to query, insert, update and modify data. Today, SQL is commonly used for Web database development and management. SQL commands can be executed when a Web page loads. This makes it possible to create dynamic Web pages that can display different information each time they load.

4. Explain PHP: PHP Stands for "Hypertext Preprocessor". PHP is a programming language that connects the front end to the back end. PHP script is embedded within a Web page along with its HTML. Before the page is sent to a user that has requested it, the Web server calls PHP to interpret and perform the operations called for in the PHP script.

5. Explain MySQL table: MySQL stores all the data in tables. Regardless of the prefix, each MySQL database table consists of rows and columns. The columns specify the type of data, whereas the rows contain the actual data itself. (Rows are horizontal, Columns are vertical)
